#### 安装

```
pip install git+https://github.com/MaxBai6/My_first_program.git
```
#### 更新

还没弄更新，可以在C:\python\Lib\site-packages\中找到这个包，删除，然后按照安装步骤重新下最新版本。（估计暂时没人会下我的包第二次:(）

#### 使用简介

```
import My_first_program
x=My_first_program.add_two(2)

result：x=4
```
